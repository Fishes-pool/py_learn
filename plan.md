# 任务

总结python基础语法，提纲：
1. 变量和简单数据类型：如何使用变量来存储信息，以及操作最基础的数据类型：文本（字符串）与数字;
2. 列表简介，操作列表，遍历列表，元祖;
3. if 逻辑判断;
4. 字典;
5. 用户输入和while循环;
6. 函数;
7. 类;
8. 文件和异常;
9. 测试代码

# 要求

每一个提纲都需要创建一个文件夹，每个文件夹下都有一个.md文件和.py文件:
- .md文件是知识点汇总，总结每一个提纲下的语法，例如字符串常用语法，列表的几种定义语法和常用方法，对于常用和重点的语法需要醒目地标记出来；每一个知识点都需要有相关代码。
- .py文件是一个实际的demo文件，作为这块知识点的综合案例，可以选取经典的案例或者结合实际项目开发中常用的场景来设计
